Project Write-up

Knighters Music

A custom music Discord bot with rich audio quality and a variety of functions.

Jun 2025 2 min read Discord · Bot Development
Discord.js Node.js MongoDB Upstash Audio
🚧 This post is a work in progress. Features and detailed documentation will be updated soon. Until then — enjoy the rich audio quality of the bot.

Environment

Knighters Music is built on top of Discord.js, the leading Node.js library for interacting with the Discord API. For persistence and state management, the bot integrates with both MongoDB and Upstash — giving it a reliable database layer for storing server settings, queue history, and user preferences.

Commands

For a seamless user experience, Knighters Music uses Discord's native slash command system. All commands are accessible directly from the Discord chat by typing / — no need to remember prefixes or syntax.

  • Browse and autocomplete commands directly in the Discord UI
  • Clean, intuitive command names anyone can figure out instantly
  • Full command list and tutorial coming soon

Customizable Settings

One of the standout features is a built-in music equalizer with multiple presets. Whether you prefer bass-heavy beats, flat reference audio, or something in between — you can tune the sound to exactly how you like it, per server.

Intention Behind Building the Bot

The goal was simple: build one of the best music bots on Discord — one that prioritises rich audio quality and low latency above everything else. Most existing bots compromise on one or the other. Knighters Music doesn't.
